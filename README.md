# Tessel 2 Example

![Radiation](/tessel_status.jpg)

#### Install Dependencies

```
npm install -g t2-cli
```

Connect Your Tessel via USB.

#### Find Your Tessel
```
t2 list
```

#### Set Wifi

```
t2 wifi -n <network-name> -p <password>
```

#### Run Some Code

```
t2 run examples/servo.js
```
